## Netflix-Clone

and authorization.
Technologies Used: React.js, Firebase, Context-API, Tailwind CSS and TMDB API.
Live Link: https://netflixy-reacty-clone.onrender.com/
GitHub: https://github.com/adarshshan/Netflix-React
